# AWS HealthLake

Amazon HealthLake is a HIPAA-eligible, FHIR R4-compliant managed service for importing, transforming, storing, and querying health data from patients and clinical systems. It provides a transactional FHIR server with bulk import/export, integrated medical NLP for unstructured data extraction, SMART on FHIR authorization, and integration with AWS analytics services including Amazon Athena.

**Human URL:** https://aws.amazon.com/healthlake/
**Base URL:** https://healthlake.us-east-1.amazonaws.com
